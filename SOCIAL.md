## GitHub Bio

Custom AI assistant fine-tuned from SmolLM2-360M-Instruct. QLoRA training, chat-formatted data, perplexity metrics.

## Repository Description

NeuralAI is a lightweight custom language model built with PyTorch and Hugging Face transformers. Fine-tuned from SmolLM2-360M-Instruct using QLoRA for efficient training on limited hardware.
